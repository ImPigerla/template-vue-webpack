# 安装vue-cli
$ npm install -g @vue/cli  或者  yarn global add @vue/cli

## 安装init桥接工具
npm install -g @vue/cli-init  或者  yarn global add @vue/cli-init


# 安装该模版

模版支持选项
- project-name 项目名称

命令行安装

$ vue init ImPigerla/template-vue-webpack [project-name]

# 初始化
$ cd project-name
$ npm install 或者 yarn

# 开发模式
$ npm run dev

# 构建项目
$ npm run build

# 构建监听
$ npm run watch
